# Code de conduite

Ce projet adopte le [Contributor Covenant](https://www.contributor-covenant.org/fr/version/2/1/code_of_conduct/)
v2.1.

En resume : participation respectueuse et bienveillante, pas de harcelement, retours
constructifs. Tout comportement inacceptable peut etre signale au mainteneur, qui se reserve
le droit de moderer contributions et interactions.
